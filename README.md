Inspirational Quote Generator

A beautiful, responsive web application that delivers daily motivation through inspirational quotes. Built with pure HTML, CSS, and JavaScript.

Features

 50+ carefully curated inspirational quotes
 Random quote generation with smooth animations
Save your favorite quotes (local storage)
 Fully responsive design
Social media sharing (Twitter, Facebook, LinkedIn)
Modern UI with elegant transitions

Technologies Used
- HTML5
- CSS3 (with CSS Variables)
- Vanilla JavaScript
- Font Awesome Icons
- Local Storage API

 Usage
Simply open `index.html` in your browser or visit the live demo. Click "New Quote" for fresh inspiration, save your favorites, and share wisdom with others!
